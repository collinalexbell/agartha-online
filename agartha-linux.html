HackMatrix Linux

Agartha experimental desktop stack

HackMatrix Linux is a lean workstation spin built for low-latency multiplayer desktop experiments, reproducible deployments, and fast rebuilds. Pick your favorite minimal base, wire in the HackMatrix engine, and keep a predictable path from dev to production nodes.

Rolling release tuned for bleeding-edge kernels and drivers. Great when you want the freshest Mesa stack and quick patch iterations.

  • Pacman + paru for fast package syncs
  • Preset kernel config with low-latency options
  • systemd boot with minimal units enabled
  • Optional PipeWire profile for voice + stream capture

Source-first build tailored to your hardware. Ideal when you need USE-flag control and predictable ABI surfaces for native modules.

  • Clang/LLVM toolchain defaults for smaller binaries
  • Curated USE flags for SDL, Vulkan, PipeWire
  • OpenRC or systemd selectable during bootstrap
  • Binary cache option for distributed node rollouts

HackMatrix layer

Both bases ship with the HackMatrix runtime, message bus, and sample scene graphs pre-baked. A small setup script wires in GPU caps detection, installs the multi-player desktop agent, and seeds example rooms so you can test roundtrip latency immediately.

  • Seat aware agent with sandboxed IPC
  • Wayland-first, X11 fallback
  • Optional Steam Input overlay for controller testing
  • Cloud init hooks for remote capture nodes
